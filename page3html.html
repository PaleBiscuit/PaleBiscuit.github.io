HTML notes

  • HTML stands for HyperText Markup Language and is used to create the structure and content of a webpage.
  • Most HTML elements contain opening and closing tags with raw text or other HTML tags between them.
  • HTML elements can be nested inside other elements. The enclosed element is the child of the enclosing parent element.
  • Any visible content should be placed within the opening and closing <body> tags.
  • Whitespace between HTML elements helps make code easier to read while not changing how elements appear in the browser.
  • Indentation also helps make code easier to read. It makes parent-child relationships visible.
  • Comments are written in HTML using the following syntax: <!-- comment -->.
  • Tables can be split into three main sections: a head, a body, and a footer.
  • Semantic HTML introduces meaning to a page through specific elements that provide context as to what is in between the tags.
  • Semantic HTML is a modern standard and makes a website accessible for people who use screen readers to translate the webpage and improves your websites SEO.
Tag Name Description
<!DOCTYPE html> HTML declaration This declaration should always be the first line of code in your HTML files. This lets the browser know what version of HTML to expect.
<html> HTML container This element will contain all of your HTML code.
<head> Document meta data Information about the web page, like the title, belongs within the this section of the page.
<title> Tab title You can add a title to your web page by using the <title> element, inside of the head. This is also what you read in the browser tab.
<body> Body Any visible content should be placed within the opening and closing of this task
<--comment--> Comment Comments are written in HTML using this syntax. This is for your developer-eyes only and will not be presented on any webpage or included in the code.
<header> Header Contributing to create the basic structure of the webpage.
<main> Main Contributing to create the basic structure of the webpage.
<nav> Navigation Contributing to create the basic structure of the webpage.
<footer> Footer Contributing to create the basic structure of the webpage.
<section> Section Defines elements in a document, such as chapters, headings, or any other area of the document with the same theme.
<article> Article Holds content that makes sense on its own such as articles, blogs, comments, etc.
<aside> Aside Contains information that is related to the main content, but not required in order to understand the dominant information.
<h1> .. <h6> Headings and sub-headings <h1> to <h6> tags, are used to provide titles for sections of content.
<p> Paragraph Specify text.
<a> Anchor Anchor tags are used to link to internal pages, external pages or content on the same page. You can create sections on a webpage and jump to them using <a> tags and adding id's to the elements you wish to jump to.
<span> Span Specify text or blocks.
<div> Division Specify a block of text.
<em> Emphasize This is used to emphasize text.
<strong> Strong This is used to emphasize text.
<br> Break Used to create line breaks in the content.
<ol> Ordered List Create a numbered list.
<ul> Unordered List Create a bullet list.
<li> Line Create a line within either numbered or bullet list.
<table> Table Create a table.
<th> Table Head Table headings clarify the meaning of data in the table.
<tr> Table Row Add a row to the table.
<td> Table Data Add data to a row in the table.
colspan Column Span Table data can span columns using the colspan attribute.
rowspan Row Span Table data can span rows using the rowspan attribute.
<thead> Table Header A tables head is created with this.
<tbody> Table Body A tables body is created with this.
<tfoot> Table Footer A tables footer is created with this.
<figure> Figure Encapsulates all types of media.
<figcaption> Figure Caption is used to describe the media in <figure>.
<img> Image A tag that allows you to insert images through links. Remember <src> to link to location of the image you want to use and <alt> to write an alternative text in case the user cannot see the image.
<video> Video A tag that allows you to insert videos through links. Remember <src> to link to location of the image you want to use and <controls> to enable the control functions on the page for the user.
<embed> Embed Old html element that are used for media files.
<audio> Audio Element for audio files. Remember <src> to link to location of the image you want to use and <controls> to enable the control functions on the page for the user.
